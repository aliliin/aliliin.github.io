Redis 实现简单的限流

发表于 2020-02-11
本文字数： 1.1k 阅读时长 ≈ 1 分钟

利用 Redis 有序集合实现，并用管道加速。

假设 $period 秒内，一个用户只能访问 $maxCount 次。用户ID作为 key，毫秒时间戳作为 score 和 value。

一个请求进入，

  • 加入有序集合——zadd
  • 移除时间窗口之前的行为记录，剩下的都是时间窗口内的——zremrangebyscore
  • 更新过期时间——expire
  • 获取窗口内的元素数量——zcard
  • 判断窗口内元素数量是否大于最大请求限制数（maxCount），若大于等于则拒绝请求，若小于则接受请求。
